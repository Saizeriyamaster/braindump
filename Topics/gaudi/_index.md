# アントニ・ガウディ

## 概要
アントニ・ガウディ・イ・コルネット（1852–1926）。カタルーニャ出身の建築家。有機的・自然主義的な建築スタイルで知られ、バルセロナに数多くの傑作を残した。その作品群はユネスコ世界遺産に登録されている。

## 主なテーマ
- 自然をモデルにした有機建築
- カタルーニャ・モデルニスメ（カタルーニャ近代主義）
- 信仰と建築の融合
- 構造とデザインの一体化
