# ガウディのキャリアと建築思想

## Summary
- **幼少期の病気が原点**：リウマチで歩けなかったガウディは馬に乗って移動しながら自然を観察し続けた。この経験が「自然こそ最高の建築モデル」という生涯の哲学に直結した。単なる美的影響ではなく、カテナリーアーチや樹木型柱のような構造原理として建築に組み込まれた。
- **キャリアは3段階**：①様式模索期（1878–1883）：ゴシック・ムデハルなど多様な歴史様式を吸収しながら独自スタイルを形成。②グエルとの蜜月期（1883–1910年代）：強力なパトロン、繊維王エウゼビ・グエルの支援でカサ・バトリョ、グエル公園、カサ・ミラなど傑作を連発。逆さ吊り実験で独自の構造理論を確立。③完全献身期（1914–1926）：他の仕事を断ちサグラダ・ファミリアのみに集中。事務所を敷地内に移して半地下に居住した。
- **「曲線」は美意識ではなく構造論**：鎖を逆さに吊るすと生まれるカテナリー曲線を反転させると、引張力ゼロで荷重を伝達できる最効率のアーチになる。ガウディはこれを「逆さ吊り模型」で10年かけて実験し、見た目の奇抜さとは裏腹に力学的に合理的な建築を実現した。
- **信仰・民族意識との三位一体**：建築スタイルはカトリック信仰（建築を「神への奉仕」と捉え、サグラダ・ファミリアを「石で書かれた聖書」とした）とカタルーニャ主義（スペイン国王にもカタルーニャ語で話すほどの民族的誇り）が不可分に絡み合っていた。晩年は禁欲生活に入り、貧者の服を着て死んでいった。

## キャリアの3段階

### 第1期：様式の模索（1878–1883）
バルセロナ建築学校で歴史主義・折衷主義を学ぶ。ゴシック、ムデハル（イスラム＋キリスト教混合）、ジャポニスムなど多様な様式を吸収。
卒業時の指導教官はガウディを「天才か狂人かわからない」と評した。

初仕事はレイアル広場の街灯（1878）。この才能を見出したのが繊維王エウゼビ・グエル。

**主な作品：** カサ・ビセンス（1883–1885）、グエル別荘への門扉（1884）

### 第2期：グエルとの蜜月・有機スタイルの確立（1883–1910年代）
グエルという強力なパトロンを得て、実験的な建築を次々と実現。
自然の形態（木、骨、岩、蜂の巣、貝殻）を構造に取り込む「有機建築」が確立される。

この時期に「逆さ吊り実験（Hanging Chain Model）」を開発。
→ 鎖を逆さに吊るすとカテナリー曲線（懸垂曲線）が生まれる。これを反転させると、圧縮のみで成立するアーチ形状になる。引張力なしで自立する純粋な構造。

**主な作品：**
- グエル邸（1886–1890）
- カサ・バトリョ（1904–1906）— 骨と皮膚を連想させる外壁、龍の背中のような屋根
- グエル公園（1900–1914）— 自然の岩山と人工構造物の境界を消す
- カサ・ミラ（ラ・ペドレラ、1906–1912）— 直線ゼロ、流れる石の山

### 第3期：サグラダ・ファミリアへの完全献身（1914–1926）
1914年以降、他の仕事をすべて断ちサグラダ・ファミリアのみに集中。
事務所をサグラダ・ファミリアの敷地内に移し、半地下に住む。
食事は最小限、服はみすぼらしく、信仰と建築が完全に一体化。

1926年6月7日、路面電車に轢かれ意識不明で発見。貧者と間違われ手当が遅れ、3日後に死去。
バルセロナ市民は熱狂的に追悼し、サグラダ・ファミリアの地下に埋葬された。

## 建築思想の核心

### 1. 「自然は最高の建築家」
> 「新しい建築を考え出しているのではない。自然界がすべてを考え出してしまっているじゃないか」

ガウディは自然を単なるデザインの参照先ではなく、構造原理の源泉と捉えた。
木の幹が荷重を地面に伝える仕組み、骨が応力に沿って発達する形、山の岩が安定する角度——これらすべてが数百万年の進化で最適化された「解答」だと考えた。

> 「大きな書物はいつも開かれている。我々はそれを読む努力をしなければならない。その書物とは自然だ」

### 2. 「直線は人間のもの、曲線は神のもの」
ガウディの建築に直線はほとんど存在しない。これは単なる美的好みではなく、構造哲学から来ている。
曲線のアーチ（特にカテナリーアーチと放物線アーチ）は、引張力ゼロで荷重を伝達できる最も効率的な形状。直線の梁は曲げモーメントに抵抗するために余計な材料を要する。

### 3. 構造・機能・装飾の一体化
ガウディ以前の建築では、構造（柱・壁）と装飾は別物だった。
ガウディは「構造そのものが装飾になる」ことを目指した。サグラダ・ファミリアの樹木状の柱は荷重分散の論理から生まれた形であり、同時に森の中にいるような空間体験を生む。

### 4. カタルーニャ主義と民族的誇り
19世紀後半、カタルーニャでは「ルネサンサ（文化復興運動）」が起きていた。
ガウディはカタルーニャ語しか話さず（国王にも）、カタルーニャの風土・地質・歴史を建築に込めることを使命と感じていた。

### 5. カトリック信仰と建築の融合
晩年のガウディにとって、サグラダ・ファミリアは「石で書かれた聖書」だった。
識字率の低かった時代に、信者が建物を見るだけで聖書の物語を理解できるよう、彫刻・象徴・色彩を徹底的に織り込んだ。
> 「人間のクライアントに急かされる必要はない。私のクライアントは急がない」（神のことを指している）

## スタイルを形成した主な影響

| 影響源 | 具体的な受容 |
|---|---|
| ゴシック建築 | フライングバットレス不要の自立構造への挑戦（ゴシックの限界の超克） |
| ムデハル建築 | 幾何学タイル装飾、アラベスク文様 → トレンカディス（割タイルモザイク）へ |
| 自然科学 | 骨格構造、植物の成長原理、鉱物の結晶形 |
| カテナリー曲線 | 逆さ吊り模型による構造実験 |
| ラスキン・ヴィオレ＝ル＝デュク | 構造合理主義、素材の正直さ |
| カタルーニャの風景 | モンセラット山の奇岩、岩肌の色彩 |
