# 吉田松陰

## Overview

幕末の思想家・教育者。29歳で処刑されながらも、松下村塾でわずか2年9ヶ月の教育を通じて明治維新の中核人物（伊藤博文・山縣有朋ら）を輩出した。「至誠」「草莽崛起」を説き、行動と思想を一体と見なした。

## Findings

松陰の最大の特徴は「矛盾を生きた思想家」である点。攘夷を叫びながら黒船に密航を試み、死を恐れず老中暗殺計画を自白して処刑された。この「殉教」が門下生の倒幕への決意を決定的に固め、明治政府の建設者たちを生んだ。現代では帝国主義との連続性という批判的視点も生じている。
