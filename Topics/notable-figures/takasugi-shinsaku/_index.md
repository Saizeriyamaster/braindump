# 高杉晋作（たかすぎ しんさく）

天保10年（1839）〜慶応3年（1867）。幕末長州藩の志士・軍事指導者。明治維新の礎を築いた「維新の風雲児」。

## 概要

高杉晋作は、吉田松陰の松下村塾で薫陶を受けた幕末の革命家・軍事指導者。文久3年（1863年）に身分制度を超えた混成部隊・奇兵隊を創設し、元治元年（1864年）の功山寺挙兵でわずか84人を率いて藩政府に反旗を翻した。慶応2年（1866年）の第二次長州征討では幕府軍を四戦場で撃退し、幕府権威の失墜を決定的にした。明治維新の直前、27歳で結核により没した。

**辞世の句**：「おもしろきこともなき世をおもしろく　すみなすものは心なりけり」（上の句は晋作、下の句は野村望東尼）
