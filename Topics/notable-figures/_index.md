# 偉人・著名人

## Overview

歴史・科学・芸術・思想・ビジネスなど各分野で時代を動かした人物たちを研究する。
単なる伝記にとどまらず、「なぜその人物が偉大とされるのか」「時代背景とどう絡み合ったか」「現代への影響は何か」という観点で掘り下げる。

## Key Questions

### 人物の選定と評価
- [ ] 「偉人」とは何か？歴史的評価はどのように変わってきたか？
- [ ] 分野別（科学・芸術・政治・思想・ビジネス）にどのような偉人がいるか？
- [ ] 同時代に複数の天才が生まれる「ルネサンス現象」はなぜ起きるのか？

### キャリアと思想
- [ ] 各人物のキャリアの転換点・転機はどこにあったか？
- [ ] 偉人たちに共通する思考・習慣・哲学はあるか？
- [ ] 失敗や挫折がその後の業績にどう影響したか？

### 時代との相互作用
- [ ] 時代がその人物を生んだのか、それとも人物が時代を変えたのか？
- [ ] 偉人の業績は死後にどのように再評価・再発見されたか？
- [ ] 文化・地域によって「偉人」の基準はどう異なるか？
