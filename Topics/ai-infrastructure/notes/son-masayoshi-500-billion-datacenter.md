# 孫正義の$500B（80兆円）AIデータセンター投資——規模感の比較

## Summary

- 2026年3月20日、孫正義はオハイオ州に**5000億ドル（約80兆円）**規模のAIデータセンターを建設すると発表。日米21社が参画し、電力10GW（原子力発電所10基分）を自給自足する計画。
- これは2026年の主要ハイパースケーラー5社（Amazon・Alphabet・Microsoft・Meta・Oracle）の**合計年間CapEx（$660〜690B）に匹敵する**単独プロジェクト。
- Stargate（OpenAI+SoftBank+Oracle、4年間$500B）とは**別の**プロジェクト。孫正義は事実上、二つの$500B超プロジェクトを同時に走らせている。

### Stargateとの関係

孫正義は**オハイオ計画とは別に**、Stargateプロジェクトにも深く関与している。

| | オハイオ AI計画 | Stargate Project |
|---|---|---|
| 発表 | 2026年3月 | 2025年1月 |
| 規模 | $500B（単独） | $500B（4年間） |
| 主体 | SoftBank主導・日米21社 | SoftBank＋OpenAI＋Oracle＋MGX |
| 責任 | SoftBankが財務主導 | SoftBankが財務責任、OpenAIが運営 |
| 電力 | 10GW自給自足 | 目標10GW（外部電力網依存） |
| 進捗 | 2026年内着工 | テキサス拠点稼働済み、7GW計画中 |

つまり孫正義は**2つの$500B超プロジェクト**を同時進行させており、合算すると**1兆ドル超**のAIインフラに関わっていることになる。

---

### なぜこれが「史上最大」なのか

参考として、人類史上の大型インフラ事業と比較：

| プロジェクト | 規模（インフレ調整後） |
|---|---|
| アポロ計画（NASA, 1960s） | 約$2,800億 |
| 米国州間高速道路網（1950-90年代） | 約$5,000億 |
| 三峡ダム（中国） | 約$750億 |
| **孫正義 オハイオ計画** | **$5,000億（1プロジェクト）** |

単一民間企業による単一プロジェクトとしては、確かに前例がない規模。

---

### 実現性への疑問

- **資金調達**: SoftBankの自己資本では賄えず、日本の政府系ファンドや日米の金融機関を束ねる「政商」的役割が必要
- **電力**: 10GWを自給自足するための発電所建設は、データセンター以上に時間がかかる
- **実行力**: Vision Fundの投資実績（WeWork等の失敗）を踏まえると、コミットメントと実行の間には乖離リスクがある
