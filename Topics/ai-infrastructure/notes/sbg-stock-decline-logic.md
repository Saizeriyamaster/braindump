# SBG株価下落の裏ロジック——80兆円投資発表と株価の逆説

## Summary

- 孫正義の80兆円投資発表（2026/3/20）後もSBG株（9984）は続落。「なぜポジティブニュースで株が下がるのか」には明確な構造的理由がある。
- 根本は**「SBGの株価が何で支えられているか」**を理解すると見えてくる：NAV（純資産価値）と自社株買いの2本柱。
- 巨額投資コミットメントはその2本柱を両方折る。

---

## Notes

### まず前提：SBGの株価はどう決まるか

SBGは事業会社ではなく**投資持株会社**。収益ではなく、保有資産の価値（NAV）と資本配分の期待で株価が動く。

```
SBG株価 ≈ 保有資産のNAV（時価）× 割引率 + 自社株買い期待

主な保有資産（2026年時点）:
  - Arm（半導体設計、上場済み）
  - OpenAI（出資・Stargate財務責任）
  - SoftBank Corp（国内通信子会社）
  - Vision Fund保有銘柄群
```

SBGは歴史的に**NAVに対して30〜50%のディスカウント**で取引される。そのディスカウントを縮小させるのが自社株買いの役割だった。

---

### 裏ロジック：なぜ「ポジティブニュース」が売りになるのか

#### ロジック①：自社株買いの原資が消える

SBGはこれまで大規模な**自社株買い**でNAVディスカウントを縮小し、株主に還元してきた。

しかしジェフリーズが指摘したのは：

> 「$100〜200Bの米国投資コミットメントは、自社株買いに充てられるはずだったキャッシュを食いつぶす」

投資家の視点では：
```
従来の期待：
  余剰キャッシュ → 自社株買い → NAVディスカウント縮小 → 株価↑

発表後の現実：
  余剰キャッシュ → オハイオ投資・OpenAI追加出資 → 自社株買い減少 → ディスカウント拡大 → 株価↓
```

#### ロジック②：OpenAI一点集中のリスクが可視化された

S&P Globalは2026年3月に**SBGの格付け見通しをStable → Negativeに変更**。理由：
- OpenAIへの追加$30Bコミットメント（Stargate財務責任）
- 「財務柔軟性を損なう可能性」と明示

OpenAIの競争環境は激化しており（DeepSeek、Gemini、Grok等）、AI覇権が「OpenAI一強」でなくなるリスクが高まる中で、SBGはOpenAIへの集中度を増している。

#### ロジック③：「外部資金を含め80兆円」の曖昧さ

孫正義が言う$500Bのうち、**SBGが実際にコミットする金額は不明**。「外部資金含め」という表現が：
- 強気な人：「SBGは巨大プロジェクトの司令塔になる」
- 懐疑的な人：「実態は小さく、誇大広告」

どちらに振れても確信が持てない状態は、株価にはニュートラル〜ネガティブ。

#### ロジック④：マクロ逆風との合わさり

- 中東情勢→原油高→長期金利上昇→グロース株売り
- ナスダック前週末-2%
- AI過剰投資全般への懐疑（DeepSeekショック以降のセンチメント）

---

### 整理：ポジティブニュースなのに株が下がる構造

```
【孫正義の視点】          【投資家の視点】
「史上最大の投資！」  →   「自社株買いの原資が消える」
「AI覇権を獲る」     →   「OpenAI一点張りリスクが増す」
「日米21社が集結」   →   「SBGの持ち出しが不明。本当に責任を持つのか？」
「20年で150兆円」    →   「20年後の話をされても割引きしか使えない」
```

**経営者のビジョンと株主の論理は、根本的にインセンティブが違う。**
孫正義にとっての「最大の成功」は、投資家にとっての「最大のリスク要因」になりうる。
