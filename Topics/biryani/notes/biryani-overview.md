# ビリヤニ（Biryani）概要

## Summary
- 南アジア（インド・パキスタン等）発祥の炊き込みご飯。スパイスと肉・野菜を重ねて蒸らす「ダム」調理法が特徴
- ムガル帝国時代にペルシャ料理の影響を受けて発展した宮廷料理が起源
- 地域・宗教・材料によって数十種類以上のバリエーションが存在する

## Notes

### 概要
ビリヤニ（Biryani / بریانی）は、バスマティライスとスパイス、肉（鶏・羊・牛）または野菜を層状に重ね、密閉した鍋で蒸し炊きする料理。インド・パキスタン・バングラデシュ・中東・東アフリカなど広い地域で食べられる。

### 歴史
- **起源：** ペルシャ語の「ビリャン（briyan）」＝「揚げる・炒める」が語源
- **発展：** 16〜17世紀のムガル帝国（現インド・パキスタン）で宮廷料理として洗練
- **伝播：** 商人・軍隊の移動とともに南アジア全土・中東・東アフリカへ広がった

### 調理法：ダム（Dum）
ビリヤニ最大の特徴は **「ダム（Dum）」** という調理法。
1. 米と具材を別々に下調理
2. 鍋に交互に層状に重ねる（レイヤリング）
3. 蓋を小麦粉の生地やアルミホイルで密閉
4. 弱火で蒸らし、スパイスの香りを全体に染み込ませる

### 主な種類

| 種類 | 地域 | 特徴 |
|------|------|------|
| ハイデラバーディ・ビリヤニ | インド南部 | カッチ（生肉から炊く）とパッキ（調理済み肉）の2方式 |
| ラクナウィ（アウダ）ビリヤニ | インド北部 | 繊細でマイルド、サフランが香る |
| コルカタ・ビリヤニ | 西ベンガル州 | ジャガイモ入りが特徴的 |
| カラチ・ビリヤニ | パキスタン | スパイシーで濃いめの味付け |
| マラバール・ビリヤニ | ケーララ州 | 短粒のカイマライスを使用 |
| シンディ・ビリヤニ | パキスタン南部 | トマト・ヨーグルトベースでさっぱり |
| ベジタリアン・ビリヤニ | 全インド | 肉なし、野菜・豆腐・パニール使用 |

### 主なスパイス
- 基本：クミン、コリアンダー、カルダモン、シナモン、クローブ、ベイリーフ
- 香り：サフラン（高級品）、ローズウォーター
- 辛味：チリ、ブラックペッパー
- ミックス：ビリヤニマサラ（市販のスパイスブレンド）

### 付け合わせ
- **ライタ（Raita）：** ヨーグルトにキュウリ・タマネギを混ぜたもの。ビリヤニの辛さを和らげる
- **ミルチ・カ・サラン：** 唐辛子のカレー（ハイデラバーディ定番）
- **コルマ：** マイルドなカレー

### 日本でのビリヤニ事情
- 2010年代後半からインド・南アジア料理ブームで認知度が上昇
- 東京では新大久保・西葛西・大久保などにインド・バングラデシュ系の専門店が集中
- 「ビリヤニ専門店」も登場し、カレーとは別ジャンルとして確立しつつある
