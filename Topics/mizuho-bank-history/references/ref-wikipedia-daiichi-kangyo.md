# 第一勧業銀行 — Wikipedia

## Summary

第一勧業銀行（1971年〜2002年）の成り立ち、第一銀行と日本勧業銀行の合併の経緯、および2002年のみずほグループへの再編までを解説する百科事典記事。

## Key Points

- 1971年、第一銀行（渋沢栄一が1873年に創設した日本最古の銀行）と日本勧業銀行（1897年設立の特殊銀行）が合併して誕生
- 存続会社は日本勧業銀行だが、金融機関コードは第一銀行の「0001」を使用
- 「ハートの銀行」の愛称で親しまれた
- 2000年にみずほホールディングスの完全子会社となり、2002年にみずほ銀行・みずほコーポレート銀行に再編
