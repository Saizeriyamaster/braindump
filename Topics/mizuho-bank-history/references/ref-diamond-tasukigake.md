# みずほの「たすき掛け人事・縦割り組織」問題 — ダイヤモンド・オンライン

## Summary

みずほのたすき掛け人事と縦割り組織の問題が、前身の第一勧銀（第一銀行＋日本勧業銀行の合併）から続く呪縛であることを分析した記事。

## Key Points

- 持株会社と傘下2行のトップポスト3つを旧3行で分け合う「たすき掛け人事」が慣例化
- ポストが「3の倍数」になり、旧行意識がDNAとして組織に刷り込まれた
- 第一勧銀時代から既に第一銀行と日本勧業銀行の「たすき掛け人事・縦割り組織化」が問題だった
- 行内融和を誓って発足したが、人事抗争が止むことはなかった
- 金融庁は「言うべきことを言わない、言われたことだけしかしない姿勢」を問題視
