# みずほ銀行及びみずほFGに対する行政処分 — 金融庁（2021年11月）

## Summary

2021年11月26日、金融庁がみずほ銀行・みずほフィナンシャルグループに発出した業務改善命令の内容。システム障害の真因と改善要求を記載。

## Key Points

- システム上・ガバナンス上の問題の真因として「システムに係るリスクと専門性の軽視」を指摘
- 「言うべきことを言わない、言われたことだけしかしない姿勢」を厳しく批判
- 「感度の欠如」「専門性や実態の軽視」が組織文化の問題として指摘
- 社長・坂井辰史氏ら経営トップ3名が辞任
- 業務改善命令は2024年1月に解除（約3年ぶり）
