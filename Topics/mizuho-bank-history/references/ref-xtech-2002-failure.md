# みずほ銀行発足初日に大障害、「必然」だった2002年の二重引き落とし — 日経xTECH

## Summary

2002年4月1日のみずほ銀行発足初日に発生した大規模システム障害について、技術的原因と組織的原因を詳細に分析した記事。

## Key Points

- ATM障害の直接原因は旧第一勧業銀行の対外接続系システムの修正ミス
- 口座振替の遅延約250万件の原因は、請求データの振り分け処理の滞り
- 二重引き落とし約3万件は人的ミスも加わって発生
- ATMは翌日復旧、口座振替の完全復旧は4月30日
- 3行の派閥争いによる4社ベンダー体制（富士通・日立・IBM・NTTデータ）が複雑さの根源
