# コンサルグローバル アクセンチュア急拡大の分析

## Summary

アクセンチュア日本法人の急拡大を、売上高・社員数の推移データで分析した記事。10年で売上5.5倍、5年で従業員1万人増という驚異的な成長の実態を報告。

## Key Points

- 日本売上高: 2014年1,304億円 → 2024年7,175億円（5.5倍）
- 従業員数: 2020年約15,000人 → 2025年約26,000人（+11,000人）
- 2025年目標: 売上9,000億円台
- 生成AI案件の急増とDX需要拡大が成長を牽引
- コンサル業界における「アクセンチュアの一人勝ち」構図

## Relevance

[[foreign-general-consulting]] のアクセンチュア分析の主要ソース。日本のコンサル市場における圧倒的なプレゼンスを数字で示す重要データ。
