# 核問題とJCPOA：交渉から崩壊へ

## Summary

イラン核合意（JCPOA）は2015年に締結されたが、2018年の米国離脱を経て事実上崩壊した。イランはウラン濃縮を60%まで引き上げ、2025年10月にJCPOAの正式終了（Termination Day）を迎えた。国連スナップバック制裁が復活し、2025年の核交渉は失敗に終わり、2026年2月の軍事攻撃へとエスカレートした。

## Notes

### JCPOA（包括的共同行動計画）の経緯

- **2015年7月**: イランとP5+1（米英仏中露+独）がJCPOAに合意
  - ウラン濃縮を3.67%以下に制限
  - 濃縮ウラン備蓄量を300kgに制限
  - IAEA（国際原子力機関）による査察受け入れ
  - 見返りとして経済制裁の段階的解除
- **2018年5月**: トランプ大統領（第1期）がJCPOAから離脱、「最大圧力」政策を開始
- **2021-2022年**: バイデン政権下でJCPOA復帰交渉が行われたが合意に至らず

### 核開発の現状

- イランは現在ウラン濃縮を**60%**まで引き上げ（JCPOA制限の3.67%を大幅超過）
- 60%濃縮ウランを**400kg以上**保有（2025年2月比で約50%増加）
- 60%から兵器級の90%への技術的距離は短い
- IAEAのグロッシ事務局長は「JCPOAは空の殻」と発言
- IAEAはイランが核爆弾を製造しているという証拠はないとした

### JCPOA正式終了とスナップバック制裁

- **2025年10月18日**: JCPOA採択日から10年が経過し、「Termination Day」が到来
  - 国連安保理決議2231が終了
  - イランの核問題に関する安保理の公式ファイルが閉鎖
- **2025年8月28日**: 英仏独がスナップバック・メカニズムを発動
- **2025年9月27日**: 国連制裁が正式に再適用
- 中露は制裁延期決議を提出したが、安保理で否決された

### 2025年核交渉の失敗

- **2025年3月**: トランプ大統領がハメネイ師に書簡を送り、交渉を提案
  - 軍事的結果の可能性を警告
- **2025年4月12日**: 米イラン間の核交渉が開始
  - トランプは2ヶ月の交渉期限を設定
- イランが新型弾道ミサイルを公開し、米軍基地への攻撃を示唆
- 第4回交渉（2025年5月3日、ローマ）は延期
- 計3回の交渉がすべて失敗に終わった

### 経済制裁の影響

- 世界銀行は2025年・2026年のイラン経済のマイナス成長を予測
- 消費者物価指数（CPI）は前年比**45.3%**のインフレ率
- 石油輸出収入はGDPの約25%を占める
- 中国はイラン原油を割引価格で購入を継続（国連制裁にもかかわらず）
- **2025年12月28日**: 通貨暴落と生活費高騰を背景に全国各地で抗議デモが発生
