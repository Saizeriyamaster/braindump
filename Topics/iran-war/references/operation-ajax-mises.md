# Operation Ajax (1953): The CIA's Template—and Warnings for Today

## Key Points

- アジャックス作戦はCIAの海外介入の雛形となった
- モサデグの石油国有化が直接的な引き金
- 米国企業がイラン石油利益の約40%を獲得
- 外国介入が長期的なブローバック（反動）を生む典型例
- 現代の中東政策にも教訓が適用可能
